Mac使用hexo + Github 搭建博客

使用hexo搭建GitHub博客需要安装Git，hexo.

这里使用brew安装

安装homebrew：

ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装nodejs：

brew install node

安装hexo：

sudo npm install -g hexo

在安装过程中，可能会很缓慢或者卡住们可以更换一下npm源：(亲测有效)

npm –registry http://registry.cnpmjs.org info underscore

找一处地址创建自己的博客

hexo init

不要开始就创建文件夹，否则会很慢

之后继续下面步骤

cd

npm install

hexo s

可以在本地看到静态网页

创建博客

hexo new “my_first_blog”

会对应生成一个my_first_blog.md文件，这个md文件会创建在source/_posts/的文件下。
